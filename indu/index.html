Our Robust Technology Stack

The system operates on four key layers: Device, Communication, Backend, and Frontend for comprehensive monitoring.

1. Device Layer (Sensors)

  • Part-1 Module: Measures electrical health (Current, Voltage) using ACS712 and ZMPT101B sensors
  • Part-2 Module: Monitors environmental/mechanical health (Temperature, Vibration, Humidity) using DS18B20 and MPU6050/SW-420
  • Secure device identification and authentication

2. Communication & Local Node

  • Wireless Data Transmission: NRF24L01 modules send data wirelessly to the ESP32 controller
  • Local Processing Unit (ESP32): Receives data, verifies unique IDs, and uploads readings to the cloud via Wi-Fi
  • Secure communication protocols

3. Backend & AI Platform

  • Cloud Database: Utilizes Supabase and Firebase for secure, scalable data storage
  • Historical data analysis and trend identification

4. Frontend Dashboard

  • Interactive Visualization: HTML/CSS/JavaScript for responsive frontend interface
  • Multi-Role Access: Dedicated dashboards for operators, lab managers, and policy makers
  • Real-time alerts and notifications

Real-Time Dashboard & Alert

The analyzed data and Fault Prediction are instantly pushed to the Frontend Dashboard. Users receive Visual Alerts and SMS/Email Notifications for critical issues.

4

Automatic Safety Action

If the AI model detects a Critical Fault (e.g., severe over-current), the system sends a direct signal back to the machine's local relay system.

Safety Protocol: The power supply is automatically shut down to prevent irreversible damage and ensure worker safety

Maintenance & Optimization
