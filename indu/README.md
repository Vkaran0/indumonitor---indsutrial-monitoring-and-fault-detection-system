# IoT Sensor Monitoring Dashboard

A real-time dashboard for monitoring sensor data with automatic fault detection and machine analytics.

## Features

### 1. Live Monitor
- Real-time graphs for temperature, voltage, and current
- Live sensor cards with status indicators
- Click any sensor card to view detailed graph
- Historical data table with filtering options

### 2. Report Analysis
- Machine usage statistics (daily, weekly, monthly)
- Fault history and analysis
- Mean Time Between Failures (MTBF)
- Custom date range reports

### 3. Machine Details
- AI-powered efficiency analysis
- Downtime tracking
- Electricity usage monitoring
- Maintenance requirement predictions
- Smart recommendations

### 4. Fault Settings
- Automatic fault detection mode
- Customizable sensor thresholds
- Manual fault triggering
- Real-time system status monitoring

## Setup Instructions

### 1. Prerequisites
- Modern web browser
- Internet connection
- Supabase account with your sensor database

### 2. Installation

1. Download all 4 files to the same folder:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `supabase-client.js`

2. Open `index.html` in your web browser

3. The dashboard will automatically connect to your Supabase database
